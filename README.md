# <img align="left" src="https://github.com/bubner/ZeusMonitor/blob/master/app/src/main/res/mipmap-xxxhdpi/ic_launcher.webp" width="140" /></a> Zeus Monitor

An Android app built with Jetpack Compose to calculate the distance between you and a lightning
strike by using the measured delay between the lightning flash and thunder.
<br><br><br>
Integrated with smart features to give the most accurate approximation of distance between the flash
and sound.

## Features

- Dynamic calculation of speed of sound using the best approximation with Weather APIs
- Live feedback and timing
- Streamlined Material 3 design
- Calculation and timing history
- Map overlay and radius
